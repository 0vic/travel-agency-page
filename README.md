# Travel Agency Page

A simple, semantic HTML landing page showcasing destinations, packages, and top itineraries with accessible images and captions.

## Features
- Semantic structure with clear headings and sections
- Linked packages and itineraries with `target="_blank"` for external resources
- Image galleries wrapped in `<figure>` with `<figcaption>` for accessibility
- Clean, minimal layout ready for CSS enhancement

## How to view
Open `index.html` locally in your browser, or use the live demo below.
